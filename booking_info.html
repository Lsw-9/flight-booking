Booking System Information

Important information about our booking process

Current System Status

Note: Our database system (MySQL) is currently under maintenance. We've implemented a temporary solution to demonstrate the booking flow.

The booking system is currently operating in demonstration mode. While you can go through the entire booking process, the booking data will not be stored permanently in a database. Instead, we're using a client-side solution to show you how the booking flow works.

This temporary solution includes:

  • Full flight search and selection interface
  • Complete passenger information forms
  • Interactive seat selection
  • A booking confirmation page with all your details

How to Use the Demo Booking System

  1. Search for flights by selecting your departure city, destination, and travel dates
  2. Choose from the available flight options
  3. Fill in your passenger details
  4. Select your preferred seats
  5. Review your booking on the confirmation page

Once the database is fully operational, your bookings will be stored securely, and you'll receive email confirmations for your bookings.

Technical Information

The current implementation uses the following technologies:

  • Frontend: HTML5, CSS3, JavaScript
  • Client-side data handling using URL parameters and browser storage
  • Static confirmation page instead of dynamic database entries

When our MySQL database is back online, we'll restore full functionality including:

  • Secure account creation and login
  • Persistent booking storage
  • Email notifications
  • Booking management and history
